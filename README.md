# Electronics Inventory and Warranty Analytics Dashboard

A full-stack inventory analytics dashboard developed using MongoDB, Node.js, Express.js, React, and Lovable.ai. The system is designed to manage semi-structured electronics inventory data and provide real-time analytics visualization through a modern dashboard interface.

---

# Project Overview

This project demonstrates the use of MongoDB flexible schemas for storing heterogeneous electronic product specifications such as RAM, processor, GPU, battery, storage, display type, and warranty information.

The dashboard provides:
- Inventory monitoring
- Sales analytics
- Category distribution visualization
- Low stock alerts
- Warranty expiry tracking
- Revenue analytics
